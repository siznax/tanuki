1. Use Markdown

 The body of each post can be Markdown text or HTML. See the Markdown documentation for more info.
### This is a heading * this is a list * _italic_ words * *bold* words * [link](http://example.com) to stuff This is a new paragraph. <p>HTML is <i>okay</i> too.</p>

This is a heading

  • this is a list
  • italic words
  • bold words
  • link to stuff

This is a new paragraph.

HTML is okay too.

2. Include an image

 Start your post with an image URL or use the <img> tag.
http://artrelatedblog.files.wordpress.com/2012/08/tanukiavatar2h.png Tanuki pixel art courtesy of [cavepainter](http://artrelatedblog.wordpress.com/2012/08/06/new-pixel-art-avatar/).

Tanuki pixel art courtesy of cavepainter.

3. Embed video or audio

 Copy the "embed code" from your favorite source.
<iframe width="560" height="315" src="http://www.youtube.com/embed/9bZkp7q19f0" frameborder="0" allowfullscreen>

6. Make a Youtube playlist

 Make a div with class="playlist" and populate it with anchors having id="YOUTUBE", where YOUTUBE is the video identifier from YouTube. A #viewport div will be created that will switch to the video last clicked.
<div id=my_playlist class=playlist> * <a href=# id=QTk688cXTsA>Capybara Onsen</a> * <a href=# id=r-qhj3sJ5qs>WORLD ORDER: Machine Civilization</a> * <a href=# id=itvJybdcYbI>He-Man, What's Going On?</a> </div>
  • Capybara Onsen
  • WORLD ORDER: Machine Civilization
  • He-Man, What's Going On?


5. Make an image gallery

 Add a new div with class=galleryjs and fill it with img tags. Click an image to have it fill the window in a lightbox. 
<div class=galleryjs>
<img src="http://data.ukiyo-e.org/mfa/scaled/sc208694.jpg">
</div>

5. Embed a map

 Add a Leaflet JS map div (specify class="leafletjs", lat="latitude", lon="longitude", and zoom="level") or embed a map from Open Street Map, or Google.
<div id="shiodome_station" class="leafletjs" lat="35.6625424" lon="139.759645" zoom="13" style="height:320px;">

7. Pretty-print code
<pre class=prettyprint> #!/usr/bin/python if __name__=="__main__": print "hello, world." </pre>
#!/usr/bin/python
if __name__=="__main__":
    print "hello, world."
If you want HTML tags to appear properly, you have to tell Markdown that it's a code block with a preprocessor directive and extra spaces.
<?prettyprint?> <pre class=prettyprint> <!doctype html> <head><title>hello</title></head> <body>hello</body> </pre>

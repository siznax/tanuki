Demo:

    [tanuki.siznax.net](http://tanuki.siznax.net/)

Config:

    DATABASE   = PATH_TO_DATABASE
    DEBUG      = True or False

Usage:

    cd ~/code
    fork or clone http://github.com/siznax/tanuki
    cd tanuki
    sqlite3 tanuki.db < schema.sql
    export TANUKI_CONFIG=`pwd`"/CONFIG"
    python views.py
